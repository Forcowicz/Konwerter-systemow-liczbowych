Number systems converter

How to use?

Simple number systems converter calculates a number to binary, octal, decimal and hexadecimal systems. Currently accepts only integers and positive numbers.

  1. Select a system, from which you want to convert the number
  2. Enter the number in the text field
  3. Read the result on the right panel!
